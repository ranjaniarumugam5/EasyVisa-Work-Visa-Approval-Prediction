# EasyVisa - Work Visa Approval Prediction

## Project Overview

This project aims to predict whether a US work visa application will be Certified or Denied using Machine Learning techniques.

The solution helps EasyVisa and OFLC identify applicants with a higher probability of visa approval and streamline the review process.

---

## Business Problem

The Office of Foreign Labor Certification (OFLC) processes thousands of visa applications every year. Manual review is time-consuming and resource-intensive.

The objective of this project is to build a machine learning model that can predict visa approval outcomes and support data-driven decision making.

## Final Model

### Tuned Gradient Boosting (Oversampled Data)

Performance:

- Recall ≈ 0.89
- F1 Score ≈ 0.81
- Strong Generalization Performance

---

## Key Insights

- Higher prevailing wages improve certification chances.
- Yearly wage structures have better approval rates.
- Applicants with prior job experience are more likely to be certified.
- Bachelor's and Master's degree holders show higher approval rates.
- West, South, and Northeast regions have higher certification rates.

---

## Business Recommendations

- Prioritize high-probability profiles.
- Encourage competitive wage offerings.
- Use machine learning for applicant pre-screening.
- Retrain the model periodically to adapt to market changes.
